
MSFORPSPフォルダをうまいこと（GAMEフォルダのなかにコピーするとか？）配置してから起動して下さい。

インストール方法などについては、以下のURLを参照して下さい。
http://mdxonline.dyndns.org/20111222_MoonShellForPSP_Ver100_InstallManual/

システムログを、eboot.pbp と同じフォルダの logbuf.txt に書き出すようにしました。
起動しないまたは何かバグがあったときは、止まった状態のPSP画面写真と logbuf.txt を見せて下さい。
